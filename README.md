# individual-project
Neste projeto construi um sistema de compras para uma loja de produtos vegetarianos; 
Para personalizar o template utilizei bootstrap, e w3-css. Meu foco principal se manteve nas validações dos dados que são enviados através do Formulario, então:   
* validei os espaços vazios; 
* validei a palavra-passe; 
* validei o formato do email;  
* Através de uma estrutura condicional, é enviado mensagem para o utilizador e assim os dados são validados ou não. Caso sejam validados são guardados em array's.
*  
*   Proximo passo => Criar uma API com php e enviar os dados para um Banco de dados.
********************************************************************************


In this project I built a system for a shopping store vegetarian products; To customize the template I used bootstrap, and w3-css. My main focus remained on the validation of the data that is sent through the Form then:  
* validated the empty spaces; 
* I validated the password; 
* I validated the email format;  
* Through a conditional structure (if and else) , a message is sent to the user and thus the data is validated or not. If they are validated, they are stored in array's.   
Next step => Create an API with php and send the data to a Database.
